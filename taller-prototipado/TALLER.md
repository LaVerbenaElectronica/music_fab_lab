## Taller de Prototipado
![image](/imagenes/taller-prototipado.jpg)
A continuación se describe el taller de prototipado generado como continuación del proyecto MUSIC FAB LAB, que tiene como objetivo hacer más accesible
el proceso de diseño de instrumentos musicales usando los elementos de los instrumentos de MUSIC FAB LAB. Para ello, se van a diseñar objetos 3D planos, 
que se van a conectar usando tornillos y cinta, para crear formas diferentes de pensar la instrumentación musical accesible. 
Aparte se va a experimentar con bloques de código ya creados y conexiones con sensores trill de diferentes formatos. 

Por otro lado, se ha creado una caja con dos potenciómetros y una barra trill. El objetivo de esta objeto es 
usarlo para enseñar a prototipar la parte digital, en Pure Data, con el objetivo de pensar instrumentos y trabajar
tanto la tecnología como la musicalidad y expresividad.

Los materiales para realizar el taller son: 

* 10 placas bela lists
as para usar.
* Diferentes piezas en 3d con agujeros para conectar entre si.
* Cinta de belcro para unir el sensor y la placa bela.
* Sensores trill de difernetes tipos: flex, bar, square, etc.
* Código en Pure Data usado en el guitalele o en el drummer boy.
